# ML Model Comparison

## Objective
To compare the performance of different machine learning algorithms:
- Logistic Regression
- K-Nearest Neighbors (KNN)

## Dataset

## Approach
1. Loaded dataset
2. Split into training and testing sets
3. Trained three models
4. Evaluated using accuracy
5. Compared results
